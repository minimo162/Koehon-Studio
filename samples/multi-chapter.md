# 第1章 はじめに

このオーディオブックは、複数の章にまたがる構成の動作確認用サンプルです。章ごとの分割、章単位の生成、章WAV結合をテストするのに使えます。

[pause:medium]

まずはじめに、全体の流れを簡単に紹介します。

# 第2章 背景

この資料が想定する読者は、音声による学習に慣れていない人です。そのため、文の長さや間のとり方を調整し、耳だけで理解できる構成にしています。

## 2.1 主な前提

前提として、以下の点を共有しておきます。第一に、元資料が存在すること。第二に、音声合成エンジンが利用可能であること。第三に、手元の環境で動作確認ができることです。

## 2.2 補足

補足として、章をまたぐ話題の繋がりを自然にするため、各章の冒頭で一行のブリッジを入れています。

[pause:short]

# 第3章 手順

実際の手順は次の通りです。

手順1。原稿を読み込みます。手順2。章分割を確認します。手順3。生成を開始します。手順4。生成完了後に章WAVを確認し、問題がなければ全体WAVを書き出します。

[pause:medium]

各手順のあいだにpauseを挟むことで、聞き手が内容を把握しやすくなります。

# 第4章 まとめ

今回のポイントは三つです。章分割の挙動、pauseタグの効果、そして複数章を統合したWAVの生成です。

[pause:long]
