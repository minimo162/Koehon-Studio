# tasks.md

# AI原稿対応オーディオブック作成アプリ 実装タスク

## 0. 開発方針

このタスク一覧は、Windows 向け Tauri + Svelte アプリとして、AI生成Markdown原稿を読み込み、ローカルTTSで音声化するMVPを作るための実装計画である。

MVPでは、元資料の直接解析は実装しない。外部AIで作成した Markdown / txt 原稿を入力し、章分割、チャンク分割、音声生成、WAV書き出し、プロンプトコピー機能を実装する。

## 1. プロジェクト初期化

- [ ] Tauri + Svelte + TypeScript プロジェクトを作成する。
- [ ] pnpm または npm のパッケージ管理を決定する。
- [ ] Tauri v2 の基本起動を確認する。
- [ ] Windows 開発環境で `pnpm tauri dev` が動くことを確認する。
- [ ] ESLint / Prettier を導入する。
- [ ] TypeScript strict mode を有効化する。
- [ ] 基本ディレクトリ構成を作成する。

推奨構成:

```text
src/
  lib/
  views/
  stores/
src-tauri/
backend/
```
