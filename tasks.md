# tasks.md

# AI原稿対応オーディオブック作成アプリ 実装タスク

## 0. 開発方針

このタスク一覧は、Windows 向け Tauri + Svelte アプリとして、AI生成Markdown原稿を読み込み、ローカルTTSで音声化する製品版を作るための実装計画である。

初期段階では、元資料の直接解析は実装しない。外部AIで作成した Markdown / txt 原稿を入力し、章分割、チャンク分割、音声生成、WAV書き出し、プロンプトコピー機能を実装する。

TTS推論は、ONNX Runtime を利用するネイティブ sidecar を前提とする。ユーザー環境に Python、pip、venv、Node.js、Rust などの追加ランタイム導入を要求しない。
